# aoc2021

Participating in https://adventofcode.com/2021 using Clojure

## Installation

Requires leinigen.

## Usage

    $ lein run

## Options

Usage: main <day.part> [data]

## Examples

    $ lein run -- 1.1 1 3 5 2 4 6
